C언어 알고리즘 공부

calendar_today 2024년 5월 22일
schedule 3분 읽기
person_outline Architect Admin

프로그래밍의 기초를 다지기 위해 C언어를 활용하여 다양한 알고리즘 문제를 해결했습니다. 단순한 문법 습득을 넘어, 효율적인 문제 해결 사고방식을 익히는 과정은 개발자로서의 성장에 있어 필수적인 단계였습니다.

terminal 소수 판별 (Prime Number Check)

소수 판별 알고리즘을 구현하며 시간 복잡도의 중요성을 학습했습니다. 단순히 1부터 N까지 나누는 방식에서 제곱근(sqrt)까지만 확인하는 최적화 기법을 적용하여 성능을 개선하는 과정을 경험했습니다. 

bool isPrime(int n) {
    if (n <= 1) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}

data_object 진법 변환 (Base Conversion)

10진수를 다양한 진법으로 변환하거나 그 반대의 과정을 C언어로 직접 구현해보았습니다. 특히 스택(Stack) 구조의 원리를 활용해 결과를 거꾸로 출력하는 논리를 세우며 메모리 구조에 대한 이해도를 높였습니다.

Decimal to Binary
나머지를 차례로 저장한 후 역순으로 재배치하여 2진수 값을 도출합니다.
Efficiency Focus
비트 연산자를 활용한 최적화된 변환 방식 또한 함께 연구했습니다.

학습 결과

C언어의 엄격한 타입 시스템과 수동적인 메모리 관리 환경에서 알고리즘을 구현해보는 것은 문제 해결 능력을 키우는 데 큰 도움이 되었습니다. 이러한 기초 체력은 추후 고수준 언어를 다룰 때에도 논리적인 코드를 작성하는 밑거름이 될 것입니다.
