1. 소수 판별 알고리즘 (Prime Number Check)

알고리즘의 가장 기초이면서도 효율성 고려의 시작점인 소수 판별입니다. 단순히 2부터 n-1까지 나누는 방식에서 벗어나, 제곱근까지만 확인하는 최적화 기법을 적용해 봅니다.

isPrime.c content_copy 
#include <stdio.h>
#include <stdbool.h>
#include <math.h>

bool isPrime(int n) {
    if (n <= 1) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;

    // sqrt(n) 까지만 확인하여 효율성 증대
    int limit = (int)sqrt(n);
    for (int i = 3; i <= limit; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    int num = 29;
    if (isPrime(num)) printf("%d is a prime number.", num);
    return 0;
}

2. 진법 변환 (Base Conversion)

10진수를 n진수로 변환하는 과정은 스택(Stack)의 개념을 이해하는 데 매우 효과적입니다. 나머지를 구하고 몫을 갱신하며, 결과값을 역순으로 출력하는 원리를 다룹니다.

Key Logic

  • 나머지 연산(%)을 통한 각 자리수 추출
  • 나눗셈 연산(/)을 통한 몫 업데이트
  • 재귀 함수 혹은 반복문을 통한 역순 출력 제어
